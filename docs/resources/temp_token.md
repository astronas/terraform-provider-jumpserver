# `jumpserver_temp_token` Resource

The `jumpserver_temp_token` resource creates a temporary authentication token in JumpServer. All fields are computed — the token is generated by the server upon creation.

## Example Usage

```hcl
resource "jumpserver_temp_token" "example" {}
```

## Argument Reference

No arguments are required.

## Attributes Reference

- **`id`** - The UUID of the temporary token.
- **`username`** - The associated username.
- **`secret`** - (Sensitive) The token secret.
- **`verified`** - Whether the token has been verified.
- **`is_valid`** - Whether the token is valid.
- **`date_created`** - The creation date.
- **`date_updated`** - The last update date.
- **`date_verified`** - The verification date.
- **`date_expired`** - The expiration date.
